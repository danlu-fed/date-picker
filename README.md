# laydate

# develop

``` base
git clone https://github.com/danlu-fed/laydate.git
git checkout develop
npm install
// dev base-style
npm start
// build
npm run build
// dev skin
npm run devskin
```

# how to build a skin

1. cd src/skins
2. new folder in here and rename to `skinName`
3. cd `skinName`
4. new 'laydate.less', recommended use `less`.
